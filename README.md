
# 🚀 ABC Agency – Website Institucional Premium

Um website institucional moderno, elegante e de altíssima performance, projetado especificamente para agências de marketing digital, produtoras de eventos ou assessoria artística. 

Este projeto foi estruturado utilizando tecnologias nativas (Vanilla Web) com foco em carregamento instantâneo, responsividade impecável e design minimalista voltado para a conversão de leads.

👉 **[Clique aqui para ver o projeto rodando ao vivo!](https://abc-agency-eta.vercel.app/)**

---

## ✨ Funcionalidades em Destaque

* **⚡ Performance Extrema:** Desenvolvido sem frameworks pesados, garantindo um carregamento ultra-rápido (essencial para campanhas de tráfego pago).
* **📱 100% Responsivo:** Layout adaptável para qualquer tamanho de tela (Desktop, Tablets e Smartphones).
* **🎨 Design Moderno & Premium:** Paleta de cores em estilo "Dark Mode" com tipografia marcante (`Syne` e `DM Sans`), ideal para o mercado de design, tecnologia e vertentes artísticas.
* **📂 Portfólio Dinâmico:** Sistema de filtros de categorias em tempo real e visualização de projetos detalhada através de janelas modais nativas (sem recarregamento de página).
* **✨ Scroll Reveal:** Animações fluidas e elegantes baseadas na API nativa `IntersectionObserver` conforme o usuário navega pelo site.
* **📲 Foco em Conversão:** Botão flutuante permanente integrado ao WhatsApp (com efeito de pulso) e formulário de contato inteligente com validação de campos em JavaScript.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as melhores práticas de desenvolvimento web moderno:

* **HTML5:** Estruturação semântica e acessível.
* **CSS3:** Estilização modular com o uso de variáveis globais, Flexbox, CSS Grid e animações otimizadas.
* **JavaScript (ES6+):** Manipulação dinâmica do DOM, controle de animações, modais e filtros de portfólio.
* **Vercel:** Plataforma de nuvem utilizada para deploy contínuo e distribuição de performance global.
